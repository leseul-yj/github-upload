##创建gitignore文件

+ 在项目根目录下面创建gitignore.txt文件

+ 把你需要排除的文件名保存到gitignore.txt文件

+ 在项目根目录下面按住Shift键并邮件然后选择“在此处打开命令窗口”

+ 执行命令 ren gitignore.txt .gitignore
